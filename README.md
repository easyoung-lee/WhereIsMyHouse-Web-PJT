# Happy House

🔔 본 프로젝트는 **삼성 청년 SW 아카데미** 1학기 관통 프로젝트 결과물입니다

<br><br>



## **개요**

- 부동산 매물의 다양한 서비스 제공을 목적으로 매물 실거래가, 최신 매물 정보, 관심지역 내 주택 실거래가 현황, 공지사항, Q&A 게시판 등을 제공
